ZVM — Zubair Virtual Machine

↗ GitHub

A stack-based bytecode virtual machine with a custom instruction set architecture. Includes Zembler — a two-pass assembler — a constant pool, bump-allocator heap, frame-based call stack, and Zebugger, an interactive CLI debugger with sidecar .dbg files. Fully documented with ISA.md and BYTECODE.md specs.

Zerminal — Unix Terminal Emulator

↗ GitHub

A Unix shell and terminal emulator written in C from scratch. Implements fork/exec, built-ins (cd, exit), pipes, I/O redirection, and signal handling. PTY support currently in progress. Clean Makefile build system; live on GitHub.

ZMP — Zubair Messaging Protocol

↗ GitHub

A custom binary TCP protocol with a 5-byte header, Fletcher-16 checksum, and select()-based multiplexing for multi-client support. Features private rooms via /create and /join commands. Documented in PROTOCOL.md. Planned: encryption and packaging as a shared C library callable from Python via ctypes.
